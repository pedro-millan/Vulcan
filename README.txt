VULCAN — Volcano Science
================================================

Proyecto web estático desarrollado como pieza de portfolio.
Simula el sitio oficial de VULCAN Volcano Science, una marca de cosmética
volcánica genderless nacida en La Palma, Islas Canarias.

------------------------------------------------
ESTRUCTURA DEL PROYECTO
------------------------------------------------

vulcan-web/
├── index.html          Página principal (SPA con navegación por secciones)
├── 404.html            Página de error personalizada
├── main.js             Lógica de navegación, animaciones y formulario
├── style.css           Estilos completos
└── assets/
    ├── images/         Imágenes y recursos gráficos (.png, .webp)
    └── video/          Vídeo de fondo (demo.mp4)

------------------------------------------------
SECCIONES
------------------------------------------------

- Home          Portada con hero, productos destacados y valores de marca
- Producto      Gama completa de 4 productos volcánicos
- Manifiesto    Historia, origen e identidad de la marca
- Contacto      Formulario funcional con envío real vía Web3Forms
- Privacidad    Política de privacidad
- FAQ           Preguntas frecuentes
- Trabaja       Página de empleo / únete al equipo

------------------------------------------------
FORMULARIO DE CONTACTO
------------------------------------------------

El formulario utiliza Web3Forms (https://web3forms.com) para el envío
de emails sin necesidad de backend.

------------------------------------------------
TECNOLOGÍAS
------------------------------------------------

- HTML5, CSS3, JavaScript (vanilla, sin frameworks)
- Web3Forms (envío de formulario)
- Fuentes: Google Fonts
- Hosting: GitHub Pages
